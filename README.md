Digital electronic design for the classic arcade game Tron, implemented in VHDL

Intructions: The VHDL code runs on a Nexys 2 FPGA (or with slight modification on a Nexys 3).
Connect two 4-button pmod controller to the FPGA and connect the FPGA to a VGA monitor. Then upload the bit file.

For detailed instruction, refer to Section 2.2 of the report.
